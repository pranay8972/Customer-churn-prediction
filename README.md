# &nbsp;        Customer Churn Prediction



### Machine Learning + Feature Engineering + XGBoost + Streamlit Deployment

It includes data preprocessing, EDA, feature engineering, model building, evaluation, and a deployable Streamlit web app.

**Streamlit Web App:**



* The project includes an interactive Streamlit app that predicts if a customer will churn based on user-input features.
* To run locally:

  *pip install -r requirements.txt*

  *streamlit run app/app.py*



**How the Model Works:**

1. Converts raw inputs to feature-engineered format
2. Applies one-hot encoding identical to training
3. Reorders features to match model input
4. Predicts churn probability
